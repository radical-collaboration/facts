=========================== FOLLOWING PSMSL DOCUMENTATION ADDED 05-Sep-2011 :
There seems to have been some confusion regarding the RLR factor for Zeebrugge following the introduction of the TAW datum in 1981. Previouly RLR factor 4.715m was added to the data. From the correspondence file this now appears in error,from 1981 data onwards the RLR factor has been reset to 4.735m, the data have been adjusted.
=========================== FOLLOWING PSMSL DOCUMENTATION ADDED 19-Jul-2012 :
While preparing the 2010 data for entry it was noticed that for some of the data values for 2008/9 the non Doodson filtered values had been used. These were replaced by the Doodson filtered values.
=========================== FOLLOWING PSMSL DOCUMENTATION ADDED 20-Jan-2014 :
When supplying the data for Zeebrugge for 2011/2012 the authority has told us that there has been subsidence in the area of the tidal sensor. The data has been corrected for this and the area now made stable. The area is to be monitored.
