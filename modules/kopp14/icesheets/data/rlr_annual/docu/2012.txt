New station Iles du Salut French Guiana. Country/station code 876/011.TGBM is benchmark 'A' 6.884m above TGZ.

=========================== FOLLOWING PSMSL DOCUMENTATION ADDED 05-Nov-2015 :   
This station was previously known as Iles du Salut the collective name for the three islands off French Guiana. This has now been changed to Ile Royale the name of the island on which the tide gauge is situated.
