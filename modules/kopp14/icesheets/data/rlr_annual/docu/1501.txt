=========================== FOLLOWING PSMSL DOCUMENTATION ADDED 24-Aug-2010 :
Tide gauge was re-instated in 2007. Previous TGBM was BM A 3.0m above Chart
Datum. TGBM is now IGN AMO 2.809m above Chart Datum. Chart Datum has remained
the same as confirmed by Ronan Creach email of 19.08.10. Therefore RLR factor
remains 6.500m.
=========================== FOLLOWING PSMSL DOCUMENTATION ADDED 28-Mar-2012 :
Now that there are several years of data for Pointe des Galets following its
re-installation the data 2007 onwards appears to be higher than previously.
Because of its remote location it is difficult to confirm this one way or the
other.
=========================== FOLLOWING PSMSL DOCUMENTATION ADDED 24-Sep-2014 :
Using values from the SONEL website the latitude and longitude for Pointe des Galets have been revised. The data downloaded from SONEL included additional data for the 1960's and 2000's. These have been added to the database.
